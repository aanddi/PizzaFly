### PizzaFly

<h3>Запуск проекта</h3>
<p>Сначало установить все зависимости<p>
```
yarn install
```
<p>Запустить компиляцию typescript в javascript<p>
```
yarn watch
```
<p>Запустить проект и наблюдатель nodemon<p>
```
yarn dev
```
